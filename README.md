# Inventory Management System

## Assignment Title

Designing a Static Management System Web Interface Using HTML and CSS

## Project Description

Static web interface for Inventory Management System. Helps businesses manage product inventory using HTML5 and CSS3.

## Features

- Home Page with intro and features
- Login Page with form
- Dashboard Page with summary boxes
- Products Listing Page with data table
- Navigation between pages
- Professional styling
- Alternating row colors in tables

## Pages

### Home Page (index.html)

- App title and welcome
- Navigation menu
- Key features
- How it works section
- Benefits list
- Call to action button
- Footer

### Login Page (login.html)

- Login form
- Email and password fields
- Remember me checkbox
- Submit button
- Security tips
- Footer

### Dashboard Page (dashboard.html)

- Welcome message
- Summary boxes with statistics
- Quick actions menu
- Recent activity table
- Product categories table
- System notifications
- Footer

### Products Listing Page (list.html)

- Product inventory table
- Filter section
- Detailed product info

## HTML Elements Used

- Headings (h1, h2, h3, h4)
- Paragraphs
- Lists (ul, ol, li)
- Tables
- Forms (form, input, label, button, select)
- Semantic tags (header, nav, main, section, article, footer)
- Anchor tags
- Images

## CSS Styling

The stylesheet includes:

- Typography (font-family, font-size, font-weight)
- Colors (text and background)
- Box model (padding, margin, borders)
- Layout (block and inline-block)
- Tables styling
- Form styling
- Hover effects

## Constraints Followed

- No Flexbox
- No CSS Grid
- No Position property
- No Media queries
- Uses normal document flow
- Uses block/inline-block elements
- Spacing achieved with margin and padding

## Browser Compatibility

Works on Chrome, Firefox, Edge, Safari, Opera

## How to Use

1. Open index.html in web browser
2. Use navigation menu to browse pages
3. All links are working

## Navigation

All pages linked through navigation menu:

- Home -> Login -> Dashboard -> Products
